# 함수

>> 함수는 다음과 같이 선언합니다.  
'''C
ret_type func_name(arguments);
'''
>> 함수의 이름은 함수의 포인터를 담고있으며,    
>> 함수 호출 연산 (함수이름 뒤에 괄호와  
>> 호출인자들을 넣음)을 통해서 함수를 호출할   
>> 수 있습니다.  
>> 맨 앞에 있는 ret_type은 반환 타입으로,   
>> 함수가 반환될 때 받는 타입입니다.  
>> aruguments는 함수의 인자들로,  
>> 자료형과 바인드될 이름으로 구성됩니다.  
>> 인자들은 스택 및 레지스터로 전달됩니다.
