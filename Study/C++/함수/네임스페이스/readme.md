# 네임스페이스

```C++
namespace SJ {
  int a;
  int b;
  
  void printscreen() {
    std::cout << "Hello, Screen!\n" << std::endl
  }
}
```
>> 네임스페이스는 위와 같이 선언합니다.  
>> 네임스페이스 내부의 전역변수 및 함수들은  

```C++
SJ::a = 1;
SJ::B = SJ::A+1;

SJ::printscreen();
```
>> 이런식으로 정의하거나, 호출할 수 있습니다.  

>> 클래스도 이 네임스페이스 안에 넣을 수 있습니다.  
>> 즉, 네임스페이스는 특정 스코프 안에 있는 멤버들에 대해서  
>> "소속"같은 개념을 부여하는 문법입니다.  
>> :: 연산자는 범위 지정 연산자라고 합니다.
