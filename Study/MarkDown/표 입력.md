# 표 입력

표는 이런 식으로 입력합니다.

```
| 첫 제목     | 두 번째 제목 | 세 번째 제목 |
| :---------  | :----------: | -----------:|
| 첫 행       | 데이터        |  111       |
| 두 번째 행  | **볼드1**     |  *이탤릭1* |
| 세 번쨰 행  | 이런식으로 나타냄          || 
```

| 첫 제목     | 두 번째 제목 | 세 번째 제목 |
| :---------  | :----------: | -----------:|
| 첫 행       | 데이터        |  111       |
| 두 번째 행  | **볼드1**     |  *이탤릭1* |
| 세 번쨰 행  |  이런 식으로 나타냄        ||

위에서 :을 찍은 위치는 정렬되는 위치를 의미합니다.  

```
:------ = 좌측 정렬
------: = 우측 정렬
:-----: = 가운데 정렬
------- = 기본 정렬
```
