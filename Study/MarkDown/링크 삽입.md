# 링크 삽입

링크를 삽입하는 방법은 다음과 같습니다.  

```
url주소를 그냥 입력하면 링크가 됩니다.
https://github.com/Nighthom/Files
글자에 입력하고 싶다면
[Nighthom/Files 디렉토리](https://github.com/Nighthom/Files)
이런식으로 입력하면 됩니다.
참조 링크는
[Nighthom/Files][1] & [1]: https://github.com/Nighthom/Files "내 리포지트리 디렉터리"
```

https://github.com/Nighthom/Files  
[Nighthom/Files 디렉토리](https://github.com/Nighthom/Files)  
[Nighthom/Files][1] & [1]: https://github.com/Nighthom/Files "내 리포지트리 디렉터리"
